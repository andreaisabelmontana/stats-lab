# stats-lab

Visual probability & statistics. Eight self-contained demos, one slider away from each concept.

**Live:** https://andreaisabelmontana.github.io/stats-lab/

1. Distribution playground (Normal, Binomial, Poisson, Exponential, Beta)
2. Central Limit Theorem (any source distribution → Gaussian sample-mean histogram)
3. Bayes — 1000-person grid, posterior updates as priors move
4. Confidence intervals — 100 CIs, coverage counter
5. Hypothesis test — visual α / β / power
6. Linear regression — draggable points, live OLS line
7. Monte Carlo π — animated dart-throw estimator
8. Random walks — k paths under the √t envelope

Plain HTML + canvas + KaTeX. No frameworks.

Part of the *-lab series: [calc-lab](https://github.com/andreaisabelmontana/calc-lab) · [algos-lab](https://github.com/andreaisabelmontana/algos-lab) · [sql-lab](https://github.com/andreaisabelmontana/sql-lab)
