7. Bayesian network — inference & explaining away

The course's central tool. This three-node network factorises the joint distribution along the arrows: $P(R,S,W)=P(R)\,P(S\mid R)\,P(W\mid S,R)$. Set evidence on any node (observe it true or false) and the posteriors update by exact enumeration. Try Wet grass = true: rain becomes more likely — then add Sprinkler = true and watch rain drop again. That is explaining away: a known cause discounts a competing one.
